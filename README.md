# RoomFlow - Hotel Management System

## 🚀 Project Overview

This system allows users to register, log in, browse available rooms, and make reservations. Admins can manage rooms, users, and bookings through a dedicated dashboard with full CRUD functionality.
The project was developed as part of our DBMS coursework.

---

## 💻 Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Server Environment: XAMPP

---

## ✨ Features

### 👤 User Features
User Registration & Login
Logout functionality
View available rooms
Book reservations

### 🛠 Admin Features
Admin login access
Manage rooms (Create, Read, Update, Delete)
Manage users and reservations
View reservation details

### 🗄 Database Features
Relational database design
#### Tables:
Users
Rooms
Reservations
#### Database Views:
Reservation Details View
Available Rooms Info View

---

## 🏗 System Architecture
Role-based system (Admin & Customer hierarchy)
Frontend communicates with backend using PHP
MySQL database handles all data storage
XAMPP used as local development server

---

## ⚙️ How to Run the Project
Install XAMPP
Move project folder to:

C:\xampp\htdocs\RoomFlow
Start Apache & MySQL in XAMPP
Import roomflow.sql into phpMyAdmin
Open browser and run:
http://localhost/RoomFlow
